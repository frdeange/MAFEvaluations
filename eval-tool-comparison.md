# Quick Eval Tool Comparison Summary

| Tool | Type | Who It's For | Key Capabilities | Test Methods | CI/CD Integration | Availability |
|------|------|-------------|------------------|--------------|-------------------|-------------|
| **Copilot Studio — Built-in Agent Evaluation** | Low-code, native | Makers / Citizen Devs | Structured automated testing inside Copilot Studio; test sets via CSV/manual/AI-generated; LLM-as-judge scoring | General quality (relevance, groundedness, completeness, politeness), Similarity, Compare Meaning, Exact/Partial Match, Intent recognition | Power Platform Pipelines | Public Preview (Oct 2025) |
| **Power CAT Copilot Studio Kit (Testing)** | Low-code, open-source add-on | Makers / Admins | Direct Line API testing; Excel import/export; App Insights + Dataverse integration; multi-turn tests; plan validation for generative orchestration | Response match, Topic match, Generative answers, Adaptive cards, Multi-turn, Plan validation | Power Platform Pipelines / Power Automate | GA (updated Nov 2025) |
| **Microsoft Foundry — Evaluation SDK** | Pro-code (Python) | Developers / AI Engineers | Local + cloud evaluation; evaluates Azure AI & Semantic Kernel agents; full evaluator catalog; traces per step | Intent Resolution, Tool Call Accuracy, Task Adherence, Task Completion, Task Navigation Efficiency, RAG evaluators, Safety evaluators, Custom LLM-as-judge | Azure DevOps / GitHub Actions (via SDK) | GA https://learn.microsoft.com/en-us/azure/foundry/how-to/develop/agent-evaluate-sdk |
| **Microsoft Foundry — GitHub Action** | Pro-code, CI/CD | DevOps / AI Engineers | Automated offline evaluation in CI/CD pipelines; statistical significance testing; multi-agent comparison; summary report generation | All Foundry evaluator catalog (agent, RAG, safety, general quality, OpenAI-based graders, custom) | GitHub Actions native | Public Preview |
| **Microsoft Foundry — Agents Playground (Built-in Eval)** | Portal / UI | Developers / Architects | In-portal trace-linked evaluation; quality + safety checks by toggle; real-time debugging; trace visibility into tool calls | Quality checks (default), Safety checks (toggle), Trace-linked results | None (portal only) | GA |
| **Microsoft Foundry — Continuous Evaluation** | Pro-code / Portal | AI Ops / Developers | Post-deployment production monitoring; online evaluation; alerting on performance regressions | Sampling-based online evaluation; configurable evaluators | Azure Monitor integration | Preview https://learn.microsoft.com/en-us/azure/foundry/how-to/develop/agent-evaluate-sdk |
| **Evals for Agent Interop (M365 Dev Kit)** | Open-source starter kit | Developers / Enterprise Architects | Cross-stack M365 surface testing (Email, Calendar, Teams, Docs); declarative eval specs; leaderboard comparison; governance/audit-ready | Schema adherence, Tool call correctness, AI judge (coherence, helpfulness), Multi-agent interop | Self-hosted harness (Docker/GitHub) | Open Preview (Jan 2026) |
| **M365 Copilot — Declarative Agent Evaluation (Copilot Studio)** | Low-code, native | Makers / Enterprise | Evaluation of declarative agents in M365 Copilot; multi-turn; authenticated context; AI graders; activity map visualization | Similarity, Semantic comparison, Keyword match, Custom labels, Tool/topic invocation grader, AI quality graders | Export + reporting | Coming GA: July 2026 |
| **AutoEval** | Low-code, native | Makers / Citizen Devs | Flexible and programmable output evaluation; More powerful than Copilot Studio built-in evals; **not** full workflow inspection | "Did the agent produce the expected output across a variety of test scenarios?" | | Internal Work In Progress |
| **azure-ai-evaluation / Microsoft.Extensions.AI.Evaluation** | | | | | | |
| **Agent Framework Built-in** | Pro-Code | Developers / AI Engineers | Local + cloud evaluation built into core SDK; agent and multi-agent workflow evaluation with per-agent breakdown; multi-turn conversation splitting; custom evaluator functions; zero-code-change trace evaluation; red teaming integration; self-reflection loops; lab benchmarks (GAIA, TAU2) | Local checks (keyword, tool call, custom functions), All Foundry evaluator catalog (quality, RAG, safety, tool accuracy), Red teaming (ASR scoring), Benchmarks (GAIA, TAU2) | Native Python; `raise_for_status()` for CI gating; LocalEvaluator runs offline; GitHub Actions / Azure DevOps compatible | GA (v1.0.0, April 2026) — eval APIs marked `@experimental` |
